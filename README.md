## Install
Terminal 1:
```
./install.sh
```
It will install curl, cmake, crystal, libeigen3-dev and gnuplot.

Terminal 2:
```
cd matrix
LD_LIBRARY_PATH=/usr/local/lib
export LD_LIBRARY_PATH
```

## Build
Terminal 1:
```
./build.sh
```

## Execute
Terminal 2:
```
crystal src/src/exemples/linear_regression/3_exemples_ko.cr
```
